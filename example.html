How to install

  1. First upload the CSS and Javascript files to your server.
  2. Next include the CSS file annotatedBlissV1.css in your <head> section with the following code: <link rel="stylesheet" type="text/css" href="/your/path/to/annotatedBlissV1.css"></link>.
  3. Then include the Javascript file annotatedBlissV1.js in the <head> section with the following code: <script type="text/javascript" src="your/path/to/annotatedBlissV1.js"></script>

How to use

<div class="annotated-bliss" symbolpath="symbols/" symbolformat="png">
	"Welcome"{welcome} "to"{to} "Blissymbols"{language, worldwriting}
</div>

Annotated Bliss should be written within the div tags with attributes to tell the code where to locate the images (often something like "hosted/images/") and what format they are in.

Annotated Bliss is written in the format "text"{imagename}. Note that the imagename shouldn't include the extension (.jpg, .png, .gif etc). To use multiple symbols concatenated simply provide them with a comma in between. For example "Blissymbols"{world, writing}

To include text without a symbol (for any reason) create an image which is the same height as the other symbols but is blank and use that as the associated symbol. Equivalently Bliss without text can be done in the obvious way: ""{symbol}.

Note that providing an invalid string will fire an error alert when you open the page. Also note that all Blissymbols being used must be the same height for the alignment to render correctly.
